# **App Name**: CodeArena

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth with email/password and Google Sign-In.
- Profile Management: Users can create and manage their profiles, tracking problems solved, streak, and preferred language.
- Problem Browser: Browse and filter coding problems by difficulty and tags.
- Code Editor: Built-in code editor with syntax highlighting and language selection (C++, Java, Python).
- Mock Code Execution: Phase 1 MVP code execution using pre-defined test cases and output comparison.
- Submission Results: View submission results with verdict, execution time, and memory usage.
- Global Leaderboard: Display a global leaderboard ranking users based on problems solved.

## Style Guidelines:

- Primary color: Deep sapphire (#082567), evoking a sense of royalty, intelligence, and sophistication.
- Background color: Light beige (#F5F5DC), creating a calm and unobtrusive backdrop that accentuates content.
- Accent color: Gold (#FFD700), adding depth and visual interest while harmonizing with the primary sapphire, for buttons and interactive elements.
- Headline font: 'Playfair Display' serif font for headlines and shorter content, conveying elegance and tradition.
- Body text font: 'Lato' sans-serif font for body text, ensuring readability and a clean user experience.
- Code Font: 'Source Code Pro' monospaced font for displaying code, maintaining clarity and precision.
- Intricate, detailed icons inspired by heraldry to represent problem categories and functionalities, adding a touch of historical prestige.
- Subtle animations and transitions, such as smooth fades and parallax scrolling, to provide feedback and enhance user experience, creating a polished feel.
